## Prerequisites

1. **API Key**: Get your API key from the [Dashboard](https://dashboard.ediconnect.com).
2. **Retailer Code**: You need a valid retailer code (e.g., `WALMART_US`). For testing, use `TEST_RETAILER`.

## Step 1: Authenticate

Add your API key to the `Authorization` header.

```bash
Authorization: Bearer sk_test_123456789
```

## Step 2: Send a Test Order

Send a POST request to the ingestion endpoint.

```bash
curl -X POST https://api.ediconnect.com/v1/orders/ingest \
  -H "Authorization: Bearer sk_test_123456789" \
  -H "Content-Type: application/json" \
  -d '{
    "shopify_order_id": "test_1",
    "retailer_code": "TEST_RETAILER",
    "shipping_address": {
      "address1": "123 Test Ln",
      "city": "Testville",
      "state": "NY",
      "zip": "10001"
    },
    "items": [
      { "sku": "TEST-SKU", "quantity": 10 }
    ]
  }'
```

## Step 3: Check Status

You should receive a `200 OK` response.

```json
{
  "edi_status": "queued",
  "trace_id": "evt_test_123"
}
```
