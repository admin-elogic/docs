title: "Quickstart" description: "Execute your first EDI transaction in 30 seconds."

<Steps>
<Step title="Configure Environment">
Export your API Key.
bash export STDIO_KEY=&quot;sk_test_123456789&quot; 
</Step>
<Step title="Ingest Order (850)">
Simulate a Shopify Webhook event. This will trigger the JSON -> X12 translation.

```bash
curl -X POST [https://api.stdio.io/v1/orders/ingest](https://api.stdio.io/v1/orders/ingest) \
  -H "Authorization: Bearer $STDIO_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "shopify_order_id": "test_101",
    "retailer_code": "WALMART_US",
    "shipping_address": {
      "address1": "702 SW 8th St",
      "city": "Bentonville",
      "state": "AR",
      "zip": "72712"
    },
    "items": [
      { "sku": "WM-SKU-001", "quantity": 10 }
    ]
  }'
```


</Step>
<Step title="Verify Output">
You should receive a queued status immediately.

```json
{
  "status": "queued",
  "trace_id": "evt_8f7d9a8s7d",
  "latency_ms": 45
}
```


</Step>
</Steps>

<Check>
Success: If you see a trace_id, the X12 file has been generated and queued for the AS2 pipe.
</Check>
